## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/aftb777/Python-for-beginners.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd Python-for-beginners
```

### 3️⃣ Run Python Files

Make sure Python is installed, then run:

```bash
python filename.py
```
